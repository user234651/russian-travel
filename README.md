# Путешествие по России

**Путешествие по России** — это адаптивный сайт, созданный с использованием **HTML**, **CSS** и методологии **БЭМ (Nested)**.
Проект посвящён интересным местам нашей страны и вдохновляет на открытие новых направлений для путешествий.

---

## Возможности проекта

- Современная и адаптивная вёрстка для различных устройств
- Чистая структура кода по методологии **БЭМ (Nested)**
- Использование **Prettier** для форматирования кода
- Оптимизированные изображения и подключённые шрифты
- Семантическая и доступная HTML-разметка

---

## Основная структура проекта

```bash
.
├── blocks/               # Блоки проекта по методологии БЭМ (Nested)
│   ├── header/           # Шапка сайта
│   ├── lead/
│   ├── intro/            # Вводный раздел
│   ├── photo/            # Фото-галерея
│   ├── places/           # Раздел с описанием мест
│   ├── cover/
│   ├── footer/           # Подвал сайта
│   └── page/             # Общие стили страницы
│
├── fonts/                # Шрифты Inter (woff2)
├── images/               # Изображения для всех секций
├── vendor/               # Вспомогательные стили (normalize.css)
│
├── index.html            # Главная страница сайта
├── index.css             # Главный CSS-файл с импортами блоков
├── package.json          # Конфигурация проекта и Prettier
├── package-lock.json
└── README.md
```

## Установка и использование

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/user234651/russian-travel.git
   ```

2. Перейдите в папку проекта:

   ```bash
   cd russian-travel
   ```

3. Установите зависимости (для использования Prettier):

   ```bash
   npm install
   ```

4. Откройте файл `index.html` в браузере или запустите локальный сервер (например, с помощью VS Code Live Server).

---

## Форматирование кода

Для удобства редактирования используется **Prettier**.
Чтобы автоматически отформатировать код, выполните:

```bash
npm run format
```
